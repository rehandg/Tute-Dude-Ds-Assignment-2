# Pandas Assignment

This assignment focuses on DataFrame creation and data analysis using the Pandas library.

## Tasks Performed

- Created a DataFrame containing employee information
- Displayed the first five rows of the dataset
- Generated summary statistics for:
  - Age
  - Salary
- Calculated average salary of employees in the HR department
- Added a new Bonus column
- Filtered employees between ages 25 and 30
- Grouped employees by department
- Calculated average salary for each department
- Sorted the DataFrame based on salary
- Exported the sorted data to a CSV file

## Technologies Used

- Python
- Pandas
